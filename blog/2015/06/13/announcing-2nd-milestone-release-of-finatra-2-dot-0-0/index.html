Announcing second milestone release of Finatra 2.0.0!

posted by cacoco on 2015-06-13 09:32:08 -0700

We’re pleased to announce the second milestone release of Finatra 2.0.0!

You can download the release from Maven Central.

For a full list of changes and updates see the diff here.

Features:

  • We’ve updated and added to the example projects including:
    • A Typesafe activator template for getting started with a Finatra project using Activator.
    • An example project that shows how to configure and run a Finatra application on Heroku.
    • An example Maven pom.xml in finatra-hello-world for building a Finatra application with Maven.

See the Finatra github page for more information on getting started.

– The Finatra Team
